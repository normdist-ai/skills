# 知乎技能使用示例

## 示例 1：读取专栏文章

**用户输入**：
```
读取这篇文章：https://zhuanlan.zhihu.com/p/81248091
```

**执行步骤**：
1. 使用 WebFetch 获取页面内容
2. 解析 Markdown 格式内容
3. 输出结构化结果

**输出格式**：
```json
{
  "title": "文章标题",
  "url": "原始URL",
  "content": "正文内容（Markdown格式）",
  "images": ["图片链接列表"],
  "fetchedAt": "获取时间"
}
```

## 示例 3：搜索知乎文章

**用户输入**：
```
搜索知乎关于 Hermes 应用场景的文章
```

**执行步骤**：
1. 尝试使用 WebSearch 搜索
2. 如果无知乎结果，提示用户直接提供链接
3. 如果有知乎链接，使用 WebFetch 读取内容

**预期输出**：
```
当前搜索引擎对知乎索引有限制，建议您直接在知乎网站搜索后提供文章链接，我可以帮您读取文章内容。
```
